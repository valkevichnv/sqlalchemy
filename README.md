### Prerequisites

##### Версия

```python3.6```

##### Модули 

```pytest, flask, flask_sqlalchemy, sqlalchemy, flask_marshmallow, sqlalchemy_marshmallow```

### Запуск
```sqlalchemy.py``` 
инициирует БД, добавляет в нее схему и данные из папки 
```data```

```init.py``` 
запускает REST сервер

### Unit-тестирование
Запустить тесты можно из папку 
```tests``` 
командой 
```python3 -m pytest```
